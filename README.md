# 🔖 트위터 북마크 → Notion 자동 정리

트위터(X) 북마크를 자동으로 수집하고, 카테고리별로 분류해서 Notion 데이터베이스에 저장해주는 도구입니다.
바이브 코딩으로 만든 내용입니다

---

## ✨ 무엇을 해주나요?

- 트위터 북마크를 한 번에 전부 가져옵니다
- 키워드 기반으로 카테고리를 자동 분류합니다
- Notion 데이터베이스에 저장해서 검색·필터로 관리할 수 있게 해줍니다
- 이후 실행할 때는 새로 추가된 북마크만 가져옵니다 (중복 없음)
- AI API 키가 있으면 "기타"로 분류된 항목을 한 번 더 재분류할 수 있습니다

---

|------|-----------|------|

| OpenAI 또는 Gemini API 키 | 선택 | AI 재분류 기능 사용 시에만 필요 |

## 🚀 설치 및 실행 방법

### Step 1. Python 설치

Python이 이미 설치되어 있다면 이 단계는 건너뛰세요.

1. https://www.python.org/downloads/ 접속
2. "Download Python 3.x.x" 버튼 클릭
3. 설치 파일 실행

> ⚠️ **중요:** 설치 화면 맨 아래 **"Add Python to PATH"** 체크박스를 반드시 체크하세요!


---

### Step 2. 파일 다운로드

이 페이지 오른쪽 상단 **Code → Download ZIP** 클릭 후 압축 해제

> ⚠️ **"안전하지 않은 파일" 차단 시:** 다운받은 zip 파일을 **우클릭 → 속성** → 하단 **"차단 해제"** 체크 → 확인. 이후 압축 해제하세요.

---

### Step 3. 최초 설정 (1회만)

압축 해제한 폴더 안에서 **`setup.bat`** 파일을 더블클릭하세요.

순서대로 아래 항목들을 입력하라고 안내해줍니다.

---

#### 🐦 트위터 토큰 받는 방법

> 로그인 상태를 유지하는 쿠키 값입니다. 다른 사람에게 알려주지 마세요.

1. 크롬 또는 엣지 브라우저에서 **x.com** 로그인
2. **F12** 키를 눌러 개발자 도구 열기
3. 상단 탭에서 **Application** 클릭
4. 왼쪽 메뉴 **Cookies** → **https://x.com** 클릭
5. 목록에서 **`ct0`** 를 찾아 Value 열의 값을 복사
6. 목록에서 **`auth_token`** 을 찾아 Value 열의 값을 복사

> 💡 토큰이 만료되면 (로그아웃하거나 비밀번호 변경 시) `setup.bat`을 다시 실행해서 갱신하면 됩니다.

---

#### 📝 Notion 토큰 받는 방법

1. https://www.notion.so/my-integrations 접속 (Notion 로그인 필요)
2. **"+ 새 API 통합 만들기"** 클릭

3. 이름 아무거나 입력 (예: `트위터북마크`) → **제출**
4. **"내부 통합 시크릿"** 옆 **표시** 클릭 → 값 복사


---

#### 📄 Notion 페이지 ID 받는 방법

북마크 DB를 만들 Notion 페이지가 필요합니다. 새 페이지를 만들어도 됩니다.

1. 해당 페이지를 Notion에서 열기
2. 오른쪽 상단 **`...`** 메뉴 클릭
3. **연결** → 방금 만든 통합 이름 클릭해서 연결

4. 브라우저 주소창에서 URL 확인
   ```
   https://www.notion.so/페이지이름-abc123def456abc123def456abc123de
                                    ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                    이 32자리가 페이지 ID입니다
   ```

---

### Step 4. 실행

설정이 끝났으면 이후로는 **`run.bat`** 만 더블클릭하면 됩니다.

```
╔══════════════════════════════════════════════╗
║       트위터 북마크 파이프라인               ║
╚══════════════════════════════════════════════╝

  1. 북마크 수집     (트위터에서 새 북마크 가져오기)
  2. 분류            (카테고리별 자동 분류)
  3. Notion 업로드   (Notion DB에 저장)
  4. 전체 실행       (1 → 2 → 3 한 번에)
  5. AI 재분류       (기타 항목 OpenAI/Gemini로 재분류)
  6. 전체 재분류     (분류 결과 초기화 후 처음부터 다시)
  0. 종료
```

처음이라면 **4번 (전체 실행)** 을 선택하세요. 이후에는 4번으로 주기적으로 실행하면 새 북마크만 추가됩니다.

## 🤖 AI 재분류 (선택 기능)

"기타"로 분류된 트윗이 많다면 AI가 한 번 더 분류해줄 수 있습니다.  
**OpenAI 또는 Gemini 키 중 하나만 있으면 됩니다.** 없어도 나머지 기능은 정상 동작합니다.

- OpenAI 키 발급: https://platform.openai.com/api-keys
- Gemini 키 발급: https://aistudio.google.com/app/apikey

`setup.bat`을 다시 실행하면 키 입력 항목이 나옵니다.  
또는 같은 폴더의 `.env` 파일을 메모장으로 열어 직접 추가해도 됩니다:

```
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AIza...
```

## ❓ 자주 묻는 질문

**Q. "python은(는) 내부 또는 외부 명령..."이라는 오류가 떠요**  
→ Python 설치 시 "Add Python to PATH"를 체크하지 않았습니다. Python을 제거 후 재설치하거나, 수동으로 PATH를 추가하세요.

**Q. 토큰 오류가 나요 / 북마크가 안 가져와져요**  
→ 트위터에서 로그아웃 후 재로그인하면 토큰이 바뀝니다. `setup.bat`을 다시 실행해 새 토큰을 입력하세요.

**Q. Notion에 업로드가 안 돼요**  
→ Notion 페이지에 통합(Integration)이 연결되어 있는지 확인하세요. (페이지 `...` 메뉴 → 연결)

**Q. 기존 북마크가 또 올라가요**  
→ `uploaded_ids.json` 파일이 삭제되면 업로드 기록이 사라져서 중복이 생깁니다. 이 파일을 지우지 마세요.

**Q. AI 재분류 키를 넣었는데 오류가 나요**  
→ `.env` 파일을 메모장으로 열어 `OPENAI_API_KEY` 또는 `GEMINI_API_KEY` 값이 제대로 들어갔는지 확인하세요. 앞뒤 공백이나 따옴표가 없어야 합니다.

## 📂 파일 구성

모든 파일은 **같은 폴더 안에** 있어야 합니다.

```
twitter-bookmark-notion/
├── setup.bat                  ← 최초 설정 (1회만 실행)
├── run.bat                    ← 매번 실행할 파일
├── setup_wizard.py            설정 마법사 (setup.bat이 호출)
├── run_menu.py                실행 메뉴 (run.bat이 호출)
├── bookmark_sync.py           트위터 북마크 수집
├── classify_bookmarks.py      카테고리 자동 분류
├── reclassify_ai.py           AI 재분류 (OpenAI/Gemini 선택)
├── setup_and_upload.py        Notion DB 생성 & 업로드
├── categories.json            카테고리 & 키워드 설정
│
│   ← 아래는 실행 후 자동으로 생성됩니다 (건드리지 마세요)
├── .env                       입력한 토큰 저장
├── bookmarks.jsonl            수집된 북마크
├── classified_bookmarks.jsonl 분류 완료된 북마크
├── uploaded_ids.json          Notion 업로드 기록 (지우면 중복 발생)
└── notion_db_id.txt           생성된 Notion DB ID
```

## ⚠️ 주의사항

- 이 도구는 트위터의 내부 API를 사용합니다. 트위터 정책 변경에 따라 동작이 중단될 수 있습니다.
- 트위터 토큰(`ct0`, `auth_token`)은 계정 접근 권한과 동일합니다. `.env` 파일을 타인에게 공유하거나 GitHub에 올리지 마세요.
- 개인 사용 목적으로만 사용하세요.
