# 트위터 북마크 노션에 자동 정리해주는 도구 배포합니다 🔖

트위터(X) 북마크 수천 개 쌓아두고 다시 못 찾는 분들 있죠?

저도 그래서 만들었습니다. 북마크를 한 번에 긁어와서 카테고리별로 자동 분류해서 Notion 데이터베이스에 저장해주는 도구예요.

개발 지식 없어도 사용할 수 있게 만들었습니다. 설정은 최초 1회만, 이후로는 파일 더블클릭 한 번이면 돼요.

---

## 이런 분들께 추천해요

- 트위터 북마크가 수백~수천 개인데 정리가 안 되는 분
- 나중에 찾아보려고 북마크했는데 어디 있는지 모르는 분
- 그림 팁, 연성, 공략 등 카테고리별로 분류해두고 싶은 분
- Notion 쓰고 계신 분

---

## 뭘 해주나요?

[📸 Notion DB 완성 화면 스크린샷]

- 트위터 북마크를 전부 가져옵니다
- **드로잉·채색 / 연성·팬덤 / 게임 공략 / 기술·AI / 굿즈·피규어** 등 카테고리로 자동 분류
- Notion 데이터베이스에 저장 (텍스트, 링크, 이미지 URL, 작성자 포함)
- 두 번째 실행부터는 새 북마크만 가져옴 (중복 없음)
- Gemini AI 연동 시 "기타"로 분류된 것도 재분류 가능

## 설치 방법

### 준비물

| 필요한 것 | 비고 |
|-----------|------|
| Windows PC | 필수 |
| Python | 필수 (아래 설치 방법 참고) |
| 트위터 계정 | 로그인 상태여야 함 |
| Notion 계정 | 무료 계정 가능 |

---

### 1단계. Python 설치

Python이 이미 있으신 분은 건너뛰세요.

**https://www.python.org/downloads/** 접속 → Download 버튼 클릭 → 설치 파일 실행

[📸 Python 설치 화면 스크린샷]

> ⚠️ **여기서 실수하면 안 돌아가요!**
> 설치 화면 맨 아래 **"Add Python to PATH"** 체크박스를 반드시 눌러주세요.

[📸 PATH 체크박스 위치 강조 스크린샷]

---

### 2단계. 압축 풀고 setup.bat 실행

다운받은 ZIP 압축을 풀면 폴더 안에 파일들이 있어요.
**`setup.bat`** 파일을 더블클릭하세요.

[📸 폴더 내 파일 목록 스크린샷, setup.bat 강조]

검은 창이 뜨면서 세 가지를 물어봐요. 아래 순서대로 입력하면 됩니다.

---

### 3단계. 트위터 토큰 입력

> 내 북마크에 접근하기 위한 인증 값이에요. 비밀번호 같은 거라 다른 사람한테 알려주시면 안 됩니다.

1. 크롬 또는 엣지에서 **x.com** 로그인
2. **F12** 키 누르기
3. 위쪽 탭에서 **Application** 클릭

[📸 개발자 도구 - Application 탭 스크린샷]

4. 왼쪽 **Cookies → https://x.com** 클릭
5. 목록에서 **`ct0`** 찾아서 오른쪽 Value 칸 값 복사

[📸 ct0 위치 스크린샷]

6. **`auth_token`** 도 같은 방법으로 복사
7. setup.bat 창에 각각 붙여넣기 (Ctrl+V 또는 우클릭 → 붙여넣기)

---

### 4단계. Notion 토큰 입력

1. **https://www.notion.so/my-integrations** 접속
2. **"+ 새 API 통합 만들기"** 클릭

[📸 Notion 통합 만들기 버튼 스크린샷]

3. 이름 아무거나 입력 (예: `트위터북마크`) → 제출
4. **"내부 통합 시크릿"** 옆 표시 버튼 클릭 → 복사 → setup.bat에 붙여넣기

[📸 Notion 시크릿 복사 스크린샷]

---

### 5단계. Notion 페이지 ID 입력

북마크 DB가 만들어질 Notion 페이지가 필요해요. 새 페이지를 만들어도 됩니다.

1. 해당 Notion 페이지 열기
2. 오른쪽 상단 **`...`** → **연결** → 방금 만든 통합 이름 클릭

[📸 Notion 페이지 연결 스크린샷]

3. 브라우저 주소창에서 URL 맨 끝 32자리 복사

```
https://www.notion.so/내이름/페이지이름-abc123def456abc123def456abc123de
                                         ←────────── 이 부분 ──────────→
```

[📸 URL에서 ID 위치 스크린샷]

4. setup.bat에 붙여넣기

---

### 6단계. 실행!

설정이 끝나면 이제 **`run.bat`** 만 더블클릭하면 됩니다.

[📸 run.bat 메뉴 화면 스크린샷]

**4번 (전체 실행)** 을 누르면 수집 → 분류 → Notion 업로드까지 한 번에 처리돼요.

처음 실행할 때 북마크 수에 따라 시간이 좀 걸릴 수 있어요. (1,000개 기준 약 6분)

## 자주 묻는 질문

**Q. "python은(는) 내부 또는 외부 명령..." 오류가 나요**
→ Python 설치할 때 "Add Python to PATH"를 체크 안 하셨을 거예요. Python 제거 후 다시 설치하면서 꼭 체크해주세요.

**Q. 북마크가 안 불러와져요 / 오류가 나요**
→ 트위터에서 로그아웃했거나 비밀번호를 바꾸면 토큰이 만료돼요. setup.bat을 다시 실행해서 토큰을 새로 입력해주세요.

**Q. Notion에 업로드가 안 돼요**
→ Notion 페이지에 통합 연결을 안 하셨을 수 있어요. 페이지 `...` 메뉴 → 연결에서 통합이 등록되어 있는지 확인해주세요.

**Q. 북마크가 중복으로 올라가요**
→ 폴더 안에 `uploaded_ids.json` 파일이 있어야 중복 방지가 돼요. 이 파일을 지우지 마세요.

**Q. Mac에서도 되나요?**
→ 현재는 Windows만 지원해요. 추후 Mac 버전 안내 예정입니다.
